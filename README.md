

Este projeto é um programa simples em Java que permite ao usuário inserir vários números e verificar se cada um deles é **primo ou não**.




 Permite definir quantos números serão analisados
 Verifica se um número é primo
 Utiliza otimização com raiz quadrada (`Math.sqrt`)
 Exibe o resultado para cada número informado





* Java
* Scanner (entrada de dados)
* Lógica de programação

bash

bash
javac Main.java


4. Execute o programa:

bash
java Main




 Como funciona

* O usuário informa quantos números deseja testar
* Para cada número:

  * O programa verifica se ele é menor ou igual a 1
  * Caso não seja, ele testa divisores até a raiz quadrada do número
* Se não houver divisores, o número é considerado primo
